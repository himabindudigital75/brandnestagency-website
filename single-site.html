Helpful notes

  • The repository contains many image files (PNG/JPG) used by pages. Keep the directory structure as-is so pages load images correctly.
  • Many CSS files in the repo have a .txt extension (for example style.css.txt, about.css.txt). Browsers will still load them when linked directly. If you'd prefer, rename them to .css on your server (or locally) to match standard practice.
  • Some HTML files in the repo are empty (for example services.html, pamphlet-design.html) — I left the links in place so you can open and edit them easily.
